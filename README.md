### 📝 줄거리
**극심한 환경파괴**로 더이상 지상에 어떤 생명체도 생존할 수 없는 21XX년.

인류는 **지하 도시**를 구축하여 땅 밑으로 내려갔습니다.


**계층**이 나뉘어진 지하 세계. 사람의 존귀를 **생산성**으로 나누게 된, 잿빛 죽음의 시대.

당신은 가장 깊은 곳,
**하층부 최후의 정원사**입니다.

그리고...

당신은, 이곳에서, *정원을 가꿀 겁니다.*

### 👥️ 인물 정보
- **주인공**: 상층부 EDEN의 연구원으로 일하다가, *기적의 씨앗*을 개발한다. 연구 성과를 보고하기 위해 상사의 집무실에 갔다가 지하세계의 모순을 알아채고 하층부로 도망가 씨앗을 틔워내기로 결심한다.

- **마리아**: 하층부의 *의사*. 하층부에 떨어진 주인공을 치료하고 지도자에게 안내한다.

- **엘더 김**: 하층부 작은 마을의 지도자. 마을의 총책임자로서 주인공의 거주를 결정하고 감시한다.

- **강민**: 암시장 거래인. 상-중층과의 비밀스러운 인맥으로 하층부에서 구할 수 없는 물건들을 구해온다.

- **윤서**: 하층부의 *교사*. 하층부에 남은 아이들을 가르친다.

- **준수**: 순박하고 먹을 것 좋아하는 어린 남자 아이.
